IE6 flicker fix

发表于 2011-09-23   |  

IE6 flicker Fix

通过CSS修复

html {
    filter: expression(document.execCommand("BackgroundImageCache", false, true));
}

通过JavaScript修复

<script type="text/javascript">
    try {
        document.execCommand('BackgroundImageCache', false, true);
    } catch(e) {};
</script>

更改客户端设置

Tools  >  Internet Options > Temporary Internet files > Settings
选择Automatically，确定

!important引起IE下JavaScript脚本报参数无效

发表于 2011-09-16   |  

IE6 - IE9 皆会报参数无效。

oElem.style.width = '200px !important';

或者

jQuery('#sElem').css({'width' : '200px !important'});

覆盖样式里面已经写有!important的规则，目前了解的方法只有通过元素的cssText来设置。
