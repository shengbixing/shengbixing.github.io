CSS Expression简述

CSS Expression 是微软的专有技术，只有 Windows 下的 IE 浏览器支持，其他浏览器均对 expression 表达式不识别。

受影响的浏览器

IE6 IE7 IE8(Q)。IE8标准模式已不再支持CSS Expression

常用场景

  • max-width, max-height等
  • position:fixed
